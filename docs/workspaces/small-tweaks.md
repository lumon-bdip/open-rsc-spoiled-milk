# Small Tweaks Workspace

Use this workspace for one focused fix at a time.

Allowed by default:

- Small bug fixes.
- Small content or balance edits.
- Focused tests for the touched behavior.
- Private dev server/client launches when needed.

Avoid by default:

- Broad refactors.
- Renderer migrations.
- Public hosted-server launches.
- Release updates unless explicitly requested after the fix is verified.

Before handing work back, state the exact files changed, the test or visual
check used, and whether the change is ready to merge.
