# Odds And Ends Workspace

Use this workspace for low-risk cleanup and support tasks that are not urgent
bug fixes or active feature work.

Allowed by default:

- Asset organization.
- Data audits.
- Small docs cleanup.
- Minor tooling improvements.
- Non-release packaging checks.

Avoid by default:

- Live server operations.
- Large renderer changes.
- Major gameplay systems.
- Anything that should clearly live in `small-tweaks` or a named feature
  branch.

Before handing work back, list what was cleaned, what was intentionally left
alone, and any follow-up that should become its own branch.
