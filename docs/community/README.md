# Spoiled Milk Community

Community planning and operating notes for the Spoiled Milk Discord server.

- [DISCORD-SERVER-SETUP.md](DISCORD-SERVER-SETUP.md): step-by-step server
  setup, roles, channels, permissions, and launch checklist.
- [DISCORD-POST-TEMPLATES.md](DISCORD-POST-TEMPLATES.md): copy-paste text for
  welcome posts, rules, release announcements, bug reports, balance feedback,
  suggestions, and staff triage.

Discord is the lightweight community and tester-discussion layer. GitHub
remains the official place for releases, downloads, source code, and durable
bug records.
