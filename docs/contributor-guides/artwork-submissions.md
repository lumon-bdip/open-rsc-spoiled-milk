# Artwork Submissions

Artwork submissions are reviewed before they become official game assets.

## Where Artwork Goes

New artwork starts here:

```text
submissions/artwork/inbox/
```

Each submission should have its own folder:

```text
submissions/artwork/inbox/submission-name/
```

Use the template:

```text
submissions/artwork/templates/artwork-submission-template.md
```

## Required Contents

Include:

- a `README.md` using the template
- source files, if available
- exported PNG files, if applicable
- license or permission notes
- AI-generation notes, if AI was used

## Important Rule

Submitting artwork does not automatically replace in-game assets.

The owner may accept the submission for storage, then later integrate it through
a separate implementation branch.
