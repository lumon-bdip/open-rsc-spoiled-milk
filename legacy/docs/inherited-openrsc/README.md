# Inherited OpenRSC Documentation

This directory keeps upstream OpenRSC/Cabbage setup and process documents for reference while MyWorld moves to a standalone repo shape.

For current MyWorld setup and development workflows, use the root [README.md](/home/justin/Core-Framework/README.md) and [docs/myworld](/home/justin/Core-Framework/docs/myworld).

The archived guides, command reference, contribution guide, security policy, launchers, and deployment scripts may mention old world names, old ports, upstream GitLab paths, or non-MyWorld launch paths. Treat them as inherited engine history, not current MyWorld instructions.

Archived launcher/deployment areas:

- `legacy-launchers/`: old root startup and deployment scripts.
- `legacy-server-launchers/`: old server-local launcher wrappers.
- `legacy-Makefile`: old OpenRSC/Cabbage database, backup, install, and admin recipes.
- `server-configs/`: old non-MyWorld server configuration presets.
- `sqlite-seeds/`: old non-MyWorld SQLite world seeds.
