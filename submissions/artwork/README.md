# Artwork Submissions

Artwork starts in `inbox/` and is reviewed by the owner.

## Status Folders

- `inbox/`: new artwork waiting for review.
- `accepted/`: artwork worth keeping.
- `parked/`: artwork that may fit later.
- `rejected/`: artwork the project decided not to use.
- `templates/`: reusable submission templates.

Accepted artwork is not automatically integrated into the game. Integration
should happen through a separate implementation branch.
