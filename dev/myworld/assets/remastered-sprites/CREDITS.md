# Remastered Sprite Credits

## ThatKidBobo

ThatKidBobo contributed the initial remastered sprite batch in repository
commit `abd9d08fc23cb4752d3389ed127d388af2d18b2b`. The batch contains improved
shading and coloring based on the existing game art:

- 90 entity/equipment animation frames;
- 85 inventory/ground item sprites; and
- 23 terrain/material texture images.

The workspace descriptors preserve the contributor, source commit, original
file ID, and mapping status for each promoted or staged set. Texture art and
ambiguous/incompatible images remain non-runtime until their respective review
gates are complete.

Future contributor batches should be added here when accepted, while detailed
per-set provenance stays in `set.json`.
